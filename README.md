## Shortcut Keys (Phím tắt)

#### Phím tắt mặc định
##### Windows
<kbd>Shift</kbd>+<kbd>Alt</kbd>+<kbd>Left</kbd>: Previous

<kbd>Shift</kbd>+<kbd>Alt</kbd>+<kbd>Up</kbd>: Play/Pause

<kbd>Shift</kbd>+<kbd>Alt</kbd>+<kbd>Right</kbd>: Next
##### macOS
Dùng các phím media mặc định trên macOS.
